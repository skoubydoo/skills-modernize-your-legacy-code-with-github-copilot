# Account Management COBOL Test Plan

This plan captures the current business logic and observable implementation behavior in the COBOL programs (`main.cob`, `operations.cob`, `data.cob`).

## Execution Notes

- Run from repository root with `./accountsystem`.
- For independent test execution, restart the program before each test case unless the pre-conditions specify a sequence.
- Record the observed behavior in the `Actual Result` and set `Status (Pass/Fail)` after execution with business stakeholders.
